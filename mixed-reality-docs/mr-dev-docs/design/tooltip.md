# <a name="tooltip"></a>工具提示

![工具提示](images/UX_Hero_Tooltip.jpg)

工具提示是链接到另一个控件或对象的简短描述。 它们用于在仔细检查时传达提示或附加信息。 工具提示可帮助用户了解未直接描述的物理环境中不熟悉的对象。
