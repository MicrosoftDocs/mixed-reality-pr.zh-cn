### <a name="always-render-anchored-holograms-within-3-meters-of-their-anchor"></a><span data-ttu-id="4a959-130">始终呈现定位点 3 米范围内的锚定的全息影像</span><span class="sxs-lookup"><span data-stu-id="4a959-130">Always render anchored holograms within 3 meters of their anchor</span></span>
